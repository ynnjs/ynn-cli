# Ynn command line tool

[![Greenkeeper badge](https://badges.greenkeeper.io/ynnjs/ynn-cli.svg)](https://greenkeeper.io/)

## Installation

```sh
$ npm i -g ynn-cli
```

## Usage

To generate a new Ynn app:

```sh
ynn new <app-name>
```

To generate a new Ynn app in current directory:

```sh
ynn new
```
